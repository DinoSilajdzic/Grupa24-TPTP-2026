# Grupa24-TPTP-2026
TPTP Zavrsni projekat 2025/2026-Webshop autodijelovi

## O projketu:
Ovaj projekat predstavlja web shop za prodaju autodijelova pod nazivom “AutoDijelovi Kuga”. Web stranica omogućava pregled različitih kategorija autodijelova, detaljan prikaz proizvoda i kontakt formu za korisnike. Projekat je razvijen korištenjem HTML-a, CSS-a i JavaScripta uz responzivan dizajn i interaktivne funkcije.
